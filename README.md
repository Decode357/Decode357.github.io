## ğŸš€ Deploy to GitHub Pages

1. Make sure all changes are committed and pushed to `main` branch:

```bash
git add -A
git commit -m "Update portfolio"
git push origin main
```

2. Build the project:

```bash
npm run build
```

* Generates `dist/` folder with static files

3. Deploy to GitHub Pages (branch `gh-pages`):

```bash
npm run deploy
```

* Uses `gh-pages` package to push `dist/` to `gh-pages` branch
* GitHub Pages will automatically serve the website from `gh-pages`

4. Open your site:

```
https://Decode357.github.io
```
